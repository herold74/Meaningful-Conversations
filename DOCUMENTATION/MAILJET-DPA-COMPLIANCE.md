## Data Processing Details

### What Data is Processed

Through Mailjet, the following data is processed:
- User email addresses
- User first name and last name (if provided during registration)
- Email content (registration confirmation, password reset links)
- Email delivery metadata (sent, opened, clicked)

### Email Types Sent

1. **Registration Confirmation**
   - Contains: Email verification link
   - Personal data: Email address, name (optional)

2. **Password Reset**
   - Contains: Password reset token/link
   - Personal data: Email address

3. **Newsletter** (if enabled)
   - Contains: Newsletter content
   - Personal data: Email address, name (optional)
   - Legal basis: User consent (opt-in)

### Data Location

- **Mailjet Infrastructure**: EU (Primary servers in Europe)
- **Data Residency**: European Economic Area (EEA)
- **Standard Contractual Clauses (SCCs)**: Included in DPA

### Security Measures

Mailjet provides:
- ISO 27001 certification
- GDPR-compliant data processing
- Encryption in transit and at rest (TLS 1.2+)
- Regular security audits
- EU-based infrastructure

**Documentation**: https://www.mailjet.com/security-privacy/
