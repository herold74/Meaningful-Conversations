## How Google Cloud DPA Works

Unlike some service providers that require explicit contract signing, Google Cloud's DPA:

1. **Automatically applies** to all customers with active Google Cloud accounts
2. **Is integrated** into the Google Cloud Platform Terms of Service
3. **Becomes binding** when you create and use a Google Cloud account
4. **Stays current** - always up-to-date with latest GDPR requirements

**Evidence of Coverage**:
- ✅ Active Google Cloud Platform account
- ✅ Using Gemini API service
- ✅ Downloaded official DPA document
- ✅ Billing account active (even if free tier)

Through the Gemini API, the following data is processed:

- Bot responses

- API request metadata

### Data Location

- **Primary Processing**: Google Cloud data centers (global)
